дёҖдёӘеҝ«йҖҹгҖҒеҸҜеҫ®гҖҒж—  JIT дё”жҳ“дәҺи°ғиҜ•зҡ„жңүйҷҗе…ғеә“вҖ”вҖ”е°ҶзҺ°д»Ј PyTorch зҡ„дҪҝз”ЁдҪ“йӘҢеёҰе…Ҙз§‘еӯҰи®Ўз®—гҖӮ

TensorMesh жҳҜдёҖдёӘеҹәдәҺ PyTorch жһ„е»әзҡ„жңүйҷҗе…ғж–№жі•пјҲFEMпјүеә“пјҢж—ЁеңЁдјҳйӣ…иҖҢй«ҳж•Ҳең°жұӮи§ЈеҒҸеҫ®еҲҶж–№зЁӢпјҲPDEпјүгҖӮйҖҡиҝҮдёҺ PyTorch з”ҹжҖҒеҺҹз”ҹйӣҶжҲҗпјҢе®ғеңЁдёҚзүәзүІз»Ҹе…ёжңүйҷҗе…ғдёҘи°ЁжҖ§зҡ„еүҚжҸҗдёӢпјҢжҸҗдҫӣиҮӘеҠЁеҫ®еҲҶгҖҒGPU еҠ йҖҹд»ҘеҸҠзӣҙи§Ӯзҡ„ Pythonic жҺҘеҸЈгҖӮ

ж ёеҝғдјҳеҠҝВ¶

  • GPU еҺҹз”ҹдёҺеҸҜеҫ®пјҡе®Ңе…ЁеҹәдәҺ PyTorch д»Һеә•еұӮжһ„е»әгҖӮеҸӘйңҖдёҖиЎҢд»Јз ҒеҚіеҸҜе°Ҷж•ҙдёӘжңүйҷҗе…ғе·ҘдҪңжөҒиҝҒз§»еҲ° GPUвҖ”вҖ”е…¶еҗҺз»ӯзҡ„жҜҸдёҖж¬ЎиЈ…й…ҚгҖҒжұӮи§Је’ҢжўҜеәҰи®Ўз®—йғҪдјҡиҮӘеҠЁз»§жүҝи®ҫеӨҮпјҢж— йңҖеҚ•зӢ¬зҡ„еҗҺз«ҜжҲ–ж•°жҚ®жҗ¬иҝҗжӯҘйӘӨгҖӮеҺҹз”ҹ autograd еҸҜж— зјқиҙҜз©ҝиЈ…й…ҚдёҺжұӮи§ЈпјҢд»ҺиҖҢе®һзҺ°з«ҜеҲ°з«ҜеҸҜеҫ®зҡ„ PDE жөҒж°ҙзәҝгҖӮ

  • й«ҳжҖ§иғҪеј йҮҸеҢ–иЈ…й…Қпјҡз”ұ TensorGalerkin й©ұеҠЁзҡ„е…Ёеј йҮҸеҢ– Map-Reduce з®—жі•пјҢе°ҶйҖҗеҚ•е…ғиҝҗз®—иһҚеҗҲдёәеҚ•дҪ“зҡ„ GPU еҶ…ж ёпјҢж¶ҲйҷӨдәҶ Python еұӮзә§зҡ„еҫӘзҺҜпјҢзӣёиҫғдәҺеҹәдәҺ CPU зҡ„жңүйҷҗе…ғжҠҖжңҜж ҲеёҰжқҘж•°йҮҸзә§зҡ„еҠ йҖҹгҖӮ

  • ж—  JIT дё”жҳ“дәҺи°ғиҜ•пјҡеҚіж—¶жү§иЎҢпјҲeager executionпјүпјҢж— зј–иҜ‘ејҖй”ҖгҖӮеҠЁжҖҒзҪ‘ж јгҖҒиҮӘйҖӮеә”еҠ еҜҶд»ҘеҸҠдәӨдә’ејҸе·ҘдҪңжөҒеқҮеҸҜејҖз®ұеҚіз”ЁвҖ”вҖ”жІЎжңүйҮҚж–°зј–иҜ‘зҡ„е»¶иҝҹпјҢд№ҹжІЎжңүйҡҫд»ҘзҗҶи§Јзҡ„иҝҪиёӘдҝЎжҒҜгҖӮ

  • е…Ёйқўзҡ„еҚ•е…ғдёҺзҪ‘ж јж”ҜжҢҒпјҡж”ҜжҢҒдёүи§’еҪўгҖҒеӣӣйқўдҪ“гҖҒеӣӣжЈұй”Ҙе’ҢдёүжЈұжҹұеҚ•е…ғпјҢеҸҜдёәеёёи§ҒеҮ дҪ•иҮӘеҠЁз”ҹжҲҗзҪ‘ж јпјҢе№¶дёҺ Gmsh / VTK-HDF5 е®һзҺ°ж— зјқ I/OгҖӮ

  • зҒөжҙ»зҡ„жұӮи§ЈеҷЁпјҡз”ұ torch-sla й©ұеҠЁпјҢиҝҷжҳҜжҲ‘д»¬й…ҚеҘ—зҡ„еҸҜеҫ®зЁҖз–ҸзәҝжҖ§д»Јж•°еә“гҖӮзәҝжҖ§гҖҒйқһзәҝжҖ§е’Ңзү№еҫҒеҖјжұӮи§ЈеҷЁеҸҜеңЁ CPU е’Ң GPU дёҠи·ЁеӨҡз§ҚеҗҺз«ҜиҝҗиЎҢпјҢе№¶е®Ңж•ҙж”ҜжҢҒ autogradгҖҒжү№йҮҸжұӮи§Јд»ҘеҸҠеҲҶеёғејҸеӨҡ GPU жү©еұ•гҖӮ

  • Pythonic APIпјҡз”ЁзәҜ Python зј–еҶҷиҮӘе®ҡд№үејұеҪўејҸвҖ”вҖ”ж— йңҖзӢ¬з«Ӣзҡ„ DSLпјҢд№ҹж— йңҖ form зј–иҜ‘еҷЁгҖӮеҸӘиҰҒдјҡеҶҷ PyTorchпјҢе°ұиғҪеҶҷжңүйҷҗе…ғгҖӮ
